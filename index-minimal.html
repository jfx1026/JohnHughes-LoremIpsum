the Random Lorem Ipsum Generator

by John Freeborn

This project was directly inspired by the amazing (and sadly defunct) Jurassic Park Lorem Ispum Generator. I wanted to create something like that, but with a few more bells and whistles. I also wanted to use this project as a tool to learn javascript, gitHub and to expand my skills on the front-end web development side of things.

This project is available on gitHub for you to use, modify and improve. If you make something awesome with what I started here, please share it with me @jfx1026 on Twitter. I've tried to document how this works so that anyone with a little know how can take what I've built and create their own lorem ipsum generator. If you are into Seinfeld, and you are willing to format the scripts into JSON, then you could make that happen. Maybe you are more of an X-Files nerd, go to it.
